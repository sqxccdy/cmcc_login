脚本输出值



Code枚举

| code |备注 |
| -------- | ------- | 
| 10000 | 执行正常，比如短信发送成功
| 20001 | 程序代码报错，未知错误
| 20002 | 运营商出现报错，一般出现反扒机制
| 20003 | 因为操作行为有误导致的错误
